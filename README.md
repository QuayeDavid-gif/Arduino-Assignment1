# Arduino Assignment 1 — Smart Countdown

This project is my submission for Assignment 1 in the Programming C++ for Engineers Using Arduino course at Ghana Communication Technology University (GCTU).

## What this program does

The Arduino Uno counts down from 5 to 1 using a while loop. Each number is displayed on the Serial Monitor, and the LED blinks according to the countdown value using a function called flashLED().

## Concepts demonstrated

- Variables
- Functions
- While loops
- LED control
- Serial Monitor

## How to run it

Open the .ino file in Arduino IDE, connect an Arduino Uno, upload the code, and open the Serial Monitor at 9600 baud.
